Collect a set number of samples.

Collect a sequence of ADC values.

[long description]

Parameters
channelNumChannel to configure
fsSample Frequency (Hz) must be between 100Hz and 10kHz
bufferArray to buffer data, does not bounds check buffer
numberOfSamplesnumber of samples to record must be even number > 1
Returns
0 if initialization successful, -1 if fail

Uses ADC Sample Sequencer 2 and Timer 0. Does not require call to ADC_Open()

int ADC_Status ( int  id)

returns 0 when ADC_collect finishes

Returns
0 if ADC_collect is complete, else value is the TargetCount - current SampleCount

Acknowledges completion at beginning of handler to minimize sample jitter. since max sample rate is 10KHz = 100us at 80MHz system clock, ISR likely to be relatively short.
